# Getting Started

- Clone this repo

  ```
  git clone https://github.com/rodrigo2392/react-native-dynamic-island
  cd react-native-dynamic-island
  ```

- Install dependencies

  ```
  yarn install
  or
  npm install
  ```

- Run the app

  ```
  npm run ios
  ```

- Happy hacking!
